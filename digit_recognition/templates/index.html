    This project involves the development of a digit recognition AI utilizing the MNIST dataset. The model is built using a Convolutional Neural Network (CNN) implemented with the PyTorch library. Training is conducted on the Google Colab platform, leveraging its free computational resources. The trained model is then deployed as a REST API within this application using the Django framework, enabling easy interaction and integration. This setup demonstrates the practical application of deep learning and web development to solve real-world problems in image classification.
